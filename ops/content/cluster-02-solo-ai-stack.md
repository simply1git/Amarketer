# Cluster 02 — The Solo Business AI Stack (YouTube-first)

Each item = one YouTube video package + one hub page on the site + 3–5 Pinterest pins derived from it. Formats from [youtube-faceless](../../kb/channels/youtube-faceless.md); compliance: [ftc-disclosure](../../kb/policies/ftc-disclosure.md) + [no-income-claims](../../kb/policies/no-income-claims.md).

## Standardized test protocol (per tool, free plan)

1. Build the same artifact in each tool: a working opt-in funnel (landing page → email signup → 3-email welcome sequence) for a fictional one-person brand.
2. Record: setup time, where the free plan hits walls, export/ownership of your data, deliverability basics, one thing the marketing page hides.
3. Screen-record everything — the recordings ARE the video footage.
