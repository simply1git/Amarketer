# Canva Canvassador Program

- **Commissions:** not publicly listed in 2026; historical program paid up to $36 per Canva Pro subscriber. Capture actual terms at application.
- **Application:** submit website/channel, promotion method, audience description. Creator/marketer/freelancer audiences are favored categories — our niche is a natural fit.
- **Product fit:** the free tier is generous, so target the Pro angle: brand kits, background remover, scheduled posting — the features solo founders actually pay for.
- **Sequencing:** requires a live site — apply after launch (Phase 3).
